Hi! Welcome to my github personal page.

This page will be mostly used to post information about some code that I will be doing and respective info and steps. I hope it can be somehow useful.

Currently I am working as a functional analyst in a healthcare IT company, where I am also writing my dissertation / thesis as a MSc Medical Informatics student.

As a sub-part of the dissertation, I am developing a script for versioning management of openEHR archetypes in local repositories and verification of compliance of those archetypes with the openEHR GitHub CKM, in order to help the repository owners to have all these resources updated. When this script is finished, I will publicly share it under my github account. Moreover, is possible to check a live demo of the script by clicking on the openEHR CKM compliance Script on the header of this page. It is on constant deploy, so sometimes it can be unaccessible or displaying some errors. The final v1 version will be finished in end of August.
